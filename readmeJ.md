Excel-CSV 変換ツール（Golang版） 使用方法
=========================================

はじめに
--------

   * ExcelファイルをCSVファイルへ変換するだけのソフトウェアです。
   * 本ソフトウェアは、実行ファイル単体で動作します。
   * XLSXファイルのみ対応しています。XLSファイルは対応していません。
   * パスワード付きExcelファイルに対応しています。
   * 複数シートには対応していません。
   * 64bit版Windows、32bit版Windowsの両方に対応しています。
   * EXCELやACCESSコンポーネントなどは不要です。

実行ファイルについて
--------------------

   * ウィンドウアプリケーションとコンソールアプリケーションがあります。  
     それぞれ以下の通りです。

      #### *ウィンドウアプリケーション*
         ExcelToCsvGow.exe        (64-bit アプリケーション)
         ExcelToCsvGow_x86.exe    (32-bit アプリケーション)

      #### *コンソールアプリケーション*
         ExcelToCsvGoc.exe        (64-bit アプリケーション)
         ExcelToCsvGoc_x86.exe    (32-bit アプリケーション)

使用方法（ウィンドウアプリケーション）
--------------------------------------

   1. 実行ファイル（ExcelToCsvGow.exe / ExcelToCsvGow_x86.exe）を起動してください。
   2. Excelファイルを選択するダイアログが表示されます。
   3. 区切り文字を選択してください。
   4. 必要な場合は、解除パスワードを入力してください。
   5. 「Excelファイルを選択...」をクリックして、Excelファイルを選んでください。
   6. Excelファイルと同じ場所に、同名のCSVファイルが作成されます。

使用方法（コンソールアプリケーション）
----------------------------------------------

   1. コマンドプロンプトを起動し、実行ファイルが存在しているディレクトリへ移動してください。
   2. 以下の引数を指定して、実行ファイルを起動してください。  
      - -d：CSVファイルの区切り文字（comma / tab）  
      - -f：変換するExcelファイル  
      - -p：解除パスワード  
        ```  
        例：ExcelToCsvGoc -d=comma -p=hitpass -f="C:\sample\sampleXLSX.xlsx"  
        ```  
      - -h：ヘルプ表示  
        ```  
          例：ExcelToCsvGoc -h  
        ```  
   3. Excelファイルと同じ場所に、同名のCSVファイルが作成されます。
